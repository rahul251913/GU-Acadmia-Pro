JSON Format Guide

📌 Top-level structure:
{
  "subjects": [ ...array of subject objects ],
  "semester": { "num": 2, "startISO": "2026-01-01", "endISO": "2026-05-22" },
  "holidays": [ { "iso": "2026-01-26", "name": "Republic Day" } ],
  "timetable": { "1": [...slots], "2": [...], "3": [...], "4": [...], "5": [...] }
}
📌 Each subject object:
{
  "name": "Subject Name",
  "code": "SUBJECT-CODE",
  "teacher": "Teacher Name",
  "futureDow": [2],
  "calDots": {
    "2026-01": { "p": [5,12,19], "a": [26], "o": [] },
    "2026-02": { "p": [2,9],     "a": [],    "o": [16] }
  }
}
📌 Each timetable slot:
{
  "time": "9–10 AM",
  "subj": "Subject Name",
  "teacher": "Teacher Name",
  "room": "Block B GF-005"
}
futureDow = Class days (0=Sun,1=Mon…6=Sat)
p = Present dates   a = Absent   o = Other/Leave
timetable key = 1=Mon, 2=Tue, 3=Wed, 4=Thu, 5=Fri
